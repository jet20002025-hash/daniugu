# 使用说明

## 安装步骤

1. 安装Python依赖包：
```bash
pip install -r requirements.txt
```

2. （可选）如果需要使用tushare作为备用数据源，需要：
   - 访问 https://tushare.pro/ 注册账号
   - 获取token
   - 在代码中配置token（当前主要使用akshare，无需配置）

## 运行程序

```bash
python main.py
```

## 选股条件说明

程序会根据以下8个条件进行选股：

1. **历史最大量**：过去一年内出现过历史最大成交量（日成交量）

2. **年线之上**：股价持续运行在年线（250日均线）之上，且重心稳步上移

3. **启动价识别**：
   - 连续下跌不超过2个月
   - 下跌期间股价不能跌破20月均线
   - 在月线上由小阴线（跌幅-2%以内）变成小阳线（涨幅1%-10%）或大阳线（涨幅≥20%）
   - 启动价 = 反转当天的收盘价

4. **回到启动价附近**：
   - 当前价在启动价的±3%范围内
   - 当前价不能跌破反转日的最低价

5. **近期大量成交**：最近一个交易日的成交量 ≥ 上一个交易日的2倍

6. **近期涨停**：最近10个交易日内出现过涨停

7. **月成交量最大**：过去一年内有最大月成交量

8. **月线稳步上升**：月线级别股价稳步上升

## 注意事项

1. **数据获取**：程序使用akshare获取数据，需要网络连接
2. **运行时间**：完整分析所有A股需要较长时间，建议先使用测试模式（前50只股票）
3. **数据延迟**：免费数据源可能有15分钟延迟
4. **结果导出**：选股结果可以导出为Excel文件，方便进一步分析

## 常见问题

### Q: 程序运行很慢怎么办？
A: 可以先使用测试模式，只分析前50只股票。或者考虑使用付费数据源提高速度。

### Q: 获取数据失败怎么办？
A: 检查网络连接，或者稍后重试。akshare有时会因为访问频率限制而失败。

### Q: 如何修改选股条件？
A: 可以修改 `stock_screener.py` 中的条件判断逻辑，或者修改 `technical_analysis.py` 中的参数。

## 数据来源

- **akshare**：主要数据源，提供行情、资金流向等数据
- **tushare**：备用数据源（需注册）
