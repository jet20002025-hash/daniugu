# 暴涨股票特征学习分析系统 - 使用说明

## 功能简介

这个系统可以：
1. **学习暴涨股票的特征**：分析您提供的暴涨股票，提取它们在暴涨前的共同特征
2. **识别最佳买点**：根据学习到的特征，在任意股票中寻找符合特征的买点
3. **扫描所有股票**：自动扫描所有A股，找到符合暴涨特征的股票

## 使用方法

### 方法一：Web界面（推荐）

1. **启动Web服务**：
   ```bash
   python3 surge_analysis_web.py
   ```

2. **访问界面**：
   打开浏览器访问：`http://localhost:5001`

3. **使用步骤**：
   - **第一步**：添加暴涨股票
     - 输入股票代码和名称（例如：000001 平安银行）
     - 可选：输入暴涨日期（如果不输入，系统会自动查找最近一个月涨幅最大的日期）
     - 点击"添加股票"按钮
     - 重复此步骤，至少添加2只暴涨股票
   
   - **第二步**：分析共同特征
     - 点击"开始分析"按钮
     - 系统会分析所有已添加股票的共同特征
     - 查看特征统计表，了解暴涨股票的共同特点
   
   - **第三步**：寻找买点
     - 输入要分析的股票代码和名称
     - 点击"寻找买点"按钮
     - 系统会显示该股票的所有潜在买点，按匹配度排序
   
   - **第四步**：扫描所有股票（可选）
     - 输入要扫描的股票数量（留空则扫描全部）
     - 点击"开始扫描"按钮
     - 系统会找到所有符合暴涨特征的股票

### 方法二：Python代码

```python
from surge_stock_analyzer import SurgeStockAnalyzer

# 创建分析器
analyzer = SurgeStockAnalyzer()

# 1. 添加暴涨股票
analyzer.add_surge_stock("000001", "平安银行")
analyzer.add_surge_stock("000002", "万科A")
analyzer.add_surge_stock("600000", "浦发银行")

# 2. 分析共同特征
common_features = analyzer.analyze_common_features()

# 3. 寻找特定股票的买点
buy_points = analyzer.find_best_buy_points("000001", "平安银行")
for point in buy_points[:5]:  # 显示前5个
    print(f"日期: {point['日期']}, 价格: {point['价格']:.2f}, 匹配度: {point['匹配度']:.2%}")

# 4. 扫描所有股票（可选）
candidates = analyzer.scan_all_stocks(limit=100)  # 扫描前100只
for candidate in candidates[:10]:  # 显示前10个
    print(f"{candidate['股票名称']}: 匹配度 {candidate['匹配度']:.2%}")
```

## 分析的特征

系统会分析以下特征：

1. **价格特征**
   - 价格变化率（30天内）
   - 价格相对各均线的位置（MA5, MA10, MA20, MA60）

2. **成交量特征**
   - 成交量放大倍数
   - 量价配合度

3. **技术指标**
   - 均线排列（多头/空头）
   - 波动率
   - 连续上涨/下跌天数

4. **市场表现**
   - 涨停次数
   - 相对强度
   - 距离最高/最低点的距离

## 注意事项

1. **数据要求**：
   - 需要至少2只暴涨股票才能进行特征分析
   - 每只股票需要至少30天的历史数据

2. **买点识别**：
   - 匹配度阈值：默认0.7（70%）
   - 匹配度越高，说明该买点越符合暴涨股票的特征

3. **扫描功能**：
   - 扫描所有股票需要较长时间
   - 建议先用较小的数量测试（如100只）
   - 扫描结果按匹配度从高到低排序

## 示例

假设您发现以下股票在一个月内涨幅超过2倍：
- 000001 平安银行
- 000002 万科A
- 600000 浦发银行

您可以：
1. 将这些股票添加到系统中
2. 分析它们的共同特征
3. 用这些特征去寻找其他股票的买点
4. 或者扫描所有股票，找到符合特征的股票
